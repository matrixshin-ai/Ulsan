# 울산형 성과주의 예산제도

## 제도 현황

「지방재정법」 제5조: 지자체장의 성과관리체계 구축 의무 + 성과계획서·성과보고서 작성 의무 (2016년 본격화).

**핵심 문제**: 제도 형식은 갖추었지만, 성과계획서가 예산편성 심사의 핵심 자료가 아닌 **부속 설명자료**로 취급됨.

## 현행 방식의 흐름 (실제)

```
전년도 예산 + 물가상승 + 계속사업 + 정치적 우선순위 + 부서 요구
→ 예산 배분 (성과 정보 미활용)
```

## 구조적 비판 5가지

| 비판 | 내용 |
|---|---|
| 1. 성과계획서 형식화 | 예산 배분에 실제 영향을 미치지 못함 |
| 2. 산출지표 편중 | "몇 명 지원", "몇 회 개최" — 결과가 아닌 활동을 측정 |
| 3. 성과-예산 연계 약함 | "반영할 수 있다" (임의 규정), 강제성 없음 |
| 4. 평가제도 과잉·분절 | 공무원이 보고서 생산에 에너지 소진 |
| 5. 정치적 사업 예외 | 공약사업·SOC는 정성 명분으로 지표 회피 |

## 울산 사례 — 성과지표 필요 사업

| 사업 | 현행 지표 | 필요한 결과지표 |
|---|---|---|
| DRT 시범사업 | 운행 횟수 | 평균 대기시간, 교통취약지 개선 |
| 수소버스 | 도입 대수 | 실제 운행률, 탄소감축량 |
| 관광시설 | 방문객 수 | 주변상권 매출, 재방문율, 유지관리비 |
| 청년창업 | 지원 건수 | 3년 생존율, 매출, 고용창출 |

## 전환 방안 (7단계)

1. 예산 요구서 단계부터 성과지표 의무 첨부
2. 지표 표준화 (투입→산출→결과→효율성 4유형)
3. 성과-예산 매트릭스 도입 (예산심사 핵심 자료화)
4. 예산 조정 규칙 명문화 (2년 연속 미달 → 감액 원칙)
5. 울산형 지표은행 구축 (분야별 표준지표 풀)
6. 의회 심사 방식 전환 (활동→결과 질문)
7. 공개형 재정성과 대시보드 운영
