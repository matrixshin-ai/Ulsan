# 울산 DRT (수요응답형 교통)

## 왜 울산에서 DRT인가

| 구조적 문제 | 내용 |
|---|---|
| 지하철 없음 | 광역시 7개 중 유일 → DRT가 피더 아닌 간선 보완까지 담당해야 할 위험 |
| 대중교통 분담률 11.6% | 전국 최저 (2021년) |
| 교통 사각지대 | 울주군 면지역, 산업단지 인근 저밀도 주거지, 배차 1시간+ |
| 산업도시 특수 수요 | 현대차·HD현대 교대근무 패턴 ≠ 일반 통근 시간표 |
| 고령화 | 특별교통수단 76대(2021), 수요 대비 부족 |

**예산**: 제4차 대중교통계획(2022-2026) 총 1,103억 중 DRT 127억 편성.

## DRT 효과적인 조건 vs 부적합한 조건

**효과적**: 인구밀도 낮은 외곽·농촌, 수요 분산, 교통약자 문전 이동, 피크 이외 시간대 보완.

**부적합**: 고밀도 간선 수요 대체, 정시성 핵심 장거리, 디지털 취약계층 압도적 지역.

## 울산형 교통 위계

```
외곽·농촌: DRT (농촌형/피더형)
→ 버스 지선: 생활권 연결
→ 버스 간선: 주요 축
→ KTX 울산역·태화강역·북울산역: 환승 허브
(트램: 장기 핵심 축)
```

**핵심 원칙**: DRT는 고밀도 간선 대체 불가. 간선·지선 사각지대 보완에 한정.

## DRT 유형 (울산 적용 가능)

| 유형 | 특성 | 울산 적용 구역 |
|---|---|---|
| 농촌형 | 100원 택시·마실버스 통합 | 울주군 읍·면 외곽 |
| 피더형 | 철도역 시간표 연동 배차 | KTX 울산역·북울산역·태화강역 |
| 도시 마이크로 | 가상 정류장 매칭 | 남구·중구 생활권 (장기) |
| 복지형 | 고령자·장애인 문전 이동 | 특별교통수단과 통합 |

## 벤치마킹 핵심

| 사례 | 성과 | 울산 시사점 |
|---|---|---|
| 경기 똑버스 | 만족도 83점, 17개 시·군 267대 | 통합 앱+환승 할인 연동이 핵심 |
| 세종 셔클 | 이동 데이터 전면 개방, 이응패스 통합 | 민간 플랫폼 교체 시 서비스 공백 리스크 |
| 인천 I-MOD | 대기 78분→27분 단축 | 신도시형 저밀도 공백 해소 모델 |
| 독일 ioki | 철도 연계율 90%+, 공회전 차단 | KTX 울산역 피더 연계 모델 |

## 리스크 TOP 3

1. **재정·수익성**: 낮은 탑승률 → 보조금 의존. 서비스 구역 최소 인구밀도 기준 + 성과 연동 재정 지원 필요.
2. **디지털 격차**: 고령층 앱 이용 어려움 → 콜센터 필수 병행 + 키오스크 거점.
3. **기존 교통 갈등**: 버스업체·택시업계 반발. 노선 개편 불신 이미 형성. **소통·공론화가 기술보다 선행**.

**핵심 통계**: 전세계 DRT 시범사업의 **70%가 3년 내 종료** (Currie & Fournier, 2020).

## 단계별 도입 로드맵

| 단계 | 기간 | 내용 |
|---|---|---|
| 준비 | 2025~2026 | 교통 사각지대 매핑, 조례 제정, 이해관계자 협의체 |
| 시범 | 2026~2027 | 울주군 농촌형 + KTX역 피더형 2개 파일럿 (각 5~10대) |
| 확대 | 2027~2028 | 성과 검증 후 권역별 확대, 복지형 통합 |
| 통합 | 2029~ | 울산 MaaS 앱 (DRT+버스+철도+공유자전거 통합) |

## 성과 지표 (시범사업 기준)

- 평균 대기시간: **15분 이내** 목표
- 차량 가동률: **탑승 비율 50% 이상**
- 이용자 만족도: **80점 이상**
- 1인당 운영비: 노선버스 동일 구간 대비 효율 개선 또는 동등
