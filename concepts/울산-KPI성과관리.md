# 울산시 KPI 성과관리 체계 개편 (민선 9기)

## 핵심 명제

> **"눈에 보이는 시민 편익, 숫자로 증명하는 행정"**

기존 체계는 투입(예산 집행률)·활동(추진율)·산출(참여 인원, 시설 조성 여부) 지표 중심 → **결과(시민 생활 변화)** 지표로 전환.

## 도입 배경

1. **행정 신뢰 위기**: 수의계약·반복 보조·관행 예산 논란. 성과가 공개되지 않으면 불신 고착.
2. **시민 정책수요 폭증**: 버스·돌봄·장애인 이동권·산업위기·공공의료 등 복합 요구 → 재정 우선순위 판단 기준 필요.
3. **여소야대 정치 환경**: 민주당 시장, 국민의힘 다수 시의회 → 정치적 명분만으로 예산 통과 불가. 수치 기반 설득 필수.

## 기존 체계의 문제

- 성과계획서·성과보고서, 공약관리, 주요업무계획, 개별사업 평가 → 분산되고 연결 없음
- "사업을 추진했다" 설명에 그침 → 통행시간이 줄었는지, 돌봄 공백이 해소됐는지 불명확
- 전시성 사업(건물·행사·조형물): 완공 자체가 성과로 처리됨

## 개편 방향 (6원칙)

1. 투입·활동 지표 → **결과 지표** 우대
2. 시민·시의회에 **공개 가능한 지표** 우대
3. **기준선·목표치 없는** 신규·증액 사업은 보완 요구
4. 반복 보조·행사성·위탁·대형 시설 사업은 **제로베이스 점검**
5. 성과 미달 사업은 폐지보다 **재설계** 우선
6. 성과-예산 매트릭스 + **시민 공개 대시보드** 단계적 도입

## 사업별 KPI 제시 형식

| 항목 | 내용 |
|---|---|
| 해결하려는 문제 | 시민 또는 지역의 구체적 문제 |
| 기준선 | 현재 상태 |
| 예산 요구액 | 재정 규모 |
| 핵심 KPI | 결과 중심 지표 |
| 목표치 | 올해 또는 중기 목표 |
| 단위비용 | 성과 1단위당 비용 |
| 공개 가능성 | 시민 공개 가능 여부 |
| 예산 판단 | 확대·유지·재설계·보류 |

## 구조적 의의

- 지방재정법상 성과계획서(행자부 요구)는 이미 존재 → **기존 틀을 버리는 것이 아니라 결과 중심으로 재정렬**
- 공약관리 KPI(당선인 공약 이행 점검)와 주요업무계획 KPI(실국별 연간 계획)는 별개 → 통합·연동 방향
- 예산 심의가 "정치 공방"에서 "성과 검증"으로 전환 가능한 조건 마련
