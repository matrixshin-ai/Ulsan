# 울산 산업도시 전환 정책분석 (v2 보고서)

## 보고서 기본 정보

- **작성일**: 2026-06-09
- **성격**: 정책분석 보고서 v2 배포용 확장본
- **핵심 관점**: 탈공업화 아닌 제조업 재구성 / 비전보다 실행 / 실행의 핵심은 돈(재정)과 사람(인력)

## Executive Summary — 핵심 판단

| 핵심 질문 | 보고서 판단 |
|---|---|
| 울산의 위기 원인 | 산업이 약해서 아님. 강하고 집중된 산업구조가 전환비용을 키운 것 |
| 전환 방향 | 전기차·친환경 선박·수소·고부가 화학·제조AI·로봇·피지컬 AI |
| 가장 큰 병목 | 수요·수익성·기업 투자여력·노동시장 전환·재정 제약·인력공급 체계 |
| 재정 핵심 문제 | 총액 부족보다 자유재원 제약, 미래 세입기반 회복 투자 여부 |
| 인력정책 핵심 | 청년·직업계고·지역대학·외국인 유학생·외국인 숙련공을 하나의 인력사다리로 |

## 공간 구조별 분석

| 구역 | 산업·기능 특성 | 정책 관찰 포인트 |
|---|---|---|
| 남구·온산권 | 석유화학·에너지·항만 배후 | 위험물 안전, 환경, 노후설비 |
| 동구 | 조선업·협력업체·해양산업 | 조선경기 변동, 외국인력, 숙련 정착 |
| 북구 | 자동차·주거 | 현대차 전환, 통근·인프라 |
| 울주군 | 에너지·농촌·개발제한 | 석유화학 NCC, 신재생 에너지, 개발규제 |
| 중구 | 행정·상업 | 원도심 쇠퇴, 재개발 |

## 핵심 진단: "강한 집중"이 전환을 어렵게 한다

- 울산의 문제는 산업 기반 부족이 아님
- 자동차·조선·석유화학 기반이 오래 지속 → 이 공식을 바꾸는 것이 어려움
- 과제: **탈공업화 ✗ / 제조업 재구성 ✓**

## 전환 방향 (산업별)

| 기존 산업 | 전환 방향 |
|---|---|
| 자동차 | 전기차 + 전장화 + 로봇·AI 제조전환 |
| 조선 | 친환경 선박(LNG·암모니아·수소) + 스마트야드 |
| 석유화학 | 고부가 소재 + 탄소저감(CCUS) + 순환원료 |
| 항만 | 수소·암모니아 친환경 연료 물류 + 디지털 안전관리 |
| 산단 전체 | 피지컬 AI + 제조AI로 공정 지능화 |

## 병목 분석

### 1. 수익성·사업성 병목

- 전기차 수요 비선형 성장 — 보조금·충전 인프라 의존
- 친환경 선박 연료 표준 아직 경쟁 중 (LNG vs 암모니아 vs 수소)
- 수소경제: 앵커 수요처와 수익모델 없으면 인프라 단독 투자 어려움
- 제조AI: 센터·실증사업만으로 불충분 → 불량률·설비정지 감소 등 공정성과 입증 필요

### 2. 재정 병목

- 울산시 총예산 2026년 6조 원대 중반, 일반회계 4조 원대 중반
- 법인세는 국세 → 높은 GRDP(8,519만 원/인, 전국 최상위)가 지방재정으로 환원 안 됨
- 사회복지·법정·매칭성 지출 증가 → 자유재원 감소
- **지방공기업 잠재 부담**: 울산도시공사(개발 리스크), 시설공단(운영비), 상하수도(노후 인프라·요금 현실화)

### 3. 인력 병목 (핵심 구조적 과제)

울산은 "공장을 가진 도시"에서 "사람을 키우는 도시"로 전환해야 함.

| 인력군 | 현황 | 과제 |
|---|---|---|
| 청년 | 10대·20대 유출율 3.5~4.5% | 커리어 경로 부족이 원인 |
| 직업계고 | 취업률은 높지만 울산 잔류율 낮음 | 지역 기업과 연결고리 강화 |
| 지역대학 | 외국인 유학생 정원 보충 수단화 | 중간기술인력 Pool로 재설계 필요 |
| 외국인 근로자 | E-9(단순인력)에 머묾 | E-7-4 → F-2 장기 숙련공 정착 경로 |

### 4. 중견기업 병목

- 울산의 중견기업이 없는 게 아니라 **대기업 서사에 가려진 것**
- 자동차 부품·조선 기자재·화학소재·제조AI 장비 분야 독자 R&D·해외진출 중견기업 육성 못하면 → 대기업 생산기지에 영원히 종속

## 중앙정부 정책 활용 원칙

- 공모사업 많이 받는 것보다 **울산 산업전환 포트폴리오로 조합**하는 것이 중요
- 수소도시·산단고도화·제조AI·외국인력·지역대학 지원이 따로 움직이면 각각 효과 제한
- **울산시 역할**: 중앙정부 정책을 받아오는 기관 → 조합하는 기관

## 정책 성과 측정 3단계

| 단계 | 핵심 질문 | 실패 위험 |
|---|---|---|
| 1단계 (사업 확보) | 국비·공모·센터·인프라 확보되었는가 | 사업 선정 자체를 성과로 착각 |
| 2단계 (현장 적용) | 기업 공정·산단·항만·학교·비자제도에 실제 적용되었는가 | 실증·시범사업에 머묾 |
| 3단계 (지역 정착) | 지역기업 매출·청년 커리어·외국인 숙련 정착·재정기반으로 남았는가 | 성과가 대기업 내부 또는 일회성에 그침 |

→ **울산 정책은 1단계에 머물면 안 됨**

## 6대 핵심 정책 메시지

1. 기존 산업 버리는 전환 설계 금지 — 기존 현금흐름·숙련 유지하며 재구성
2. 기술 구호보다 사업성 검증 — 수요자·가격·투자회수·인력·금융이 맞아야 성과
3. 중앙정부 정책을 조합하는 기관으로 울산시 역할 재정의
4. 재정: 총예산보다 재량재원과 장기부담 중심 분석
5. 인력: 청년·직업계고·대학·외국인을 분리 말고 하나의 숙련 사다리로
6. 성과: 사업 선정이 아닌 **불량률 감소·울산 잔류율·숙련비자 전환** 등 현장지표
