# 김상욱 당선인 경제 공약·발언 정리

## 1. 울산 위기 진단

- 울산은 단순 경기 둔화가 아닌 **쇠락·쇠퇴·소멸 위험** 도시로 규정
- **"3년이 골든타임"** — 청년 유출, 산업 쇠퇴, 도시 고립, 기득권 경직 복합 작용
- 고립 → 도시 쇠퇴 → 개인·도시·국가 동반 쇠퇴 인식

## 2. 4대 도시 비전

1. **민주 도시** — 시민이 주인, 행정은 도구
2. **개방 도시** — 도시 간 연결, 초광역 경제권
3. **제조업 AX 선도도시** — 울산 = 국가 제조업 AI 전환 테스트베드
4. **복지 도시** — 기본 생활 인프라 우선

## 3. 노동 중심 산업 AX (핵심 차별점)

### 일반 스마트제조 공약과의 차이

일반 공약: AI·로봇 도입 → 기업 생산성 향상
김상욱 구상: **기술혁신 과실의 배분 구조**에 초점

### 로봇 공동소유 모델

- 문제인식: AI·로봇이 노동을 대체할 때 이익이 자본에 집중
- 해법: **노동자·시민 공동체 출자 협동조합이 로봇 소유권 보유**
- 로봇 운용 수익 일부를 노동자·시민 공동체가 배분받는 구조
- 울산형 노사민정 대타협과 연결
- 부산일보 인터뷰: "전국 최초 모델을 울산에서 증명하겠다"

### 노동자 보호 안전망

- **직업전환 보장제**
- **숙련노동자 AI 동행사업**
- 노동특보 도입
- 고용노동부 협의체 구성

## 4. 자동차 산업 위기 인식

- 자동차 산업 이미 로봇·AI 중심으로 전환 진행 중
- 단순 조립공정 로봇 대체 임박
- R&D 거점 울산 외부 이동 가능성 우려
- 울산은 변화를 피하는 것이 아니라 **전환의 테스트 사이트**가 되어야

## 5. 산업 AX 거점 구상

### 부울경 산업 AX 실증연구단지

| 도시 | 역할 |
|---|---|
| **울산** | 제조혁신 중심 (자동차·조선·석유화학 기반) |
| 부산 | 물류 기능 |
| 경남 | 첨단기계 산업 |

### UNIST·울산대·울산과학대 연구벨트

- 대학·연구기관 연결 → 연구개발이 실제 생산현장 적용으로
- UNIST: 부울경 AI·AX 허브 역할 (울주군 간담회 언급)

### 광주·울산 분업

- **광주**: 기초 AI·LLM 개발 / 국가AI데이터센터 (88.5 PF GPU)
- **울산**: AI 모델 산업현장 적용·실증·제조업 전환
- → [[references/초광역-AI-AX]] 공약으로 구체화

## 6. 부울경 메가시티 및 개방 도시론

- 울산 단독 생존론 거부 → 초광역 경제권 속 울산 역할 재정립
- 부산(물류)·경남(첨단기계)·울산(제조혁신) 기능 결합
- 광역교통망 확충

## 7. 재정 운영 원칙

- 치적사업·전시행정 타파 — 예산 한정, 오페라하우스(5,000억)에 쓰면 대중교통·복지·보육 예산 없어짐
- 시의회 3분의 2 야당 구도: 추경·예산 동의 불확실 → 국비 사업 우선 활용 전략
