# 민선 9기 인수위원회 업무보고 (2026-06-17~18)

## 경제산업실 (2026-06-17)

### 석유화학 위기 대응
- **산업위기선제대응지역** 지정 완료, 4,000억 원 규모 지원 신청
- S-OIL 샤힌 프로젝트 준공 시 나프타 160만 톤 추가 공급 → SK지오센트릭·대한유화 가동 위기 예고
- 감산 압박(330만 톤): SK·S-OIL·대한유화 이해관계 충돌 중
- 당선인 지시: 노동자 고용불안 해소 위한 재교육 방안, 기업별 전환 대책 세부 수립

### 에너지 로드맵
- **부유식 해상풍력 1단계(500MW)**: 전 세계 미검증 기술 + 높은 유지비 우려. 당선인이 강한 신중론
- **울산형 에너지 장기 로드맵** 7월 중순까지 수립 지시 (수소·태양광·원전·해상풍력 포트폴리오)
- **전기료 지역차등제**: 원전 소재 울산 혜택 활용, AI 데이터센터 유치에 적극 활용
- 고정식 해상풍력의 공유수면 점용료 과다 문제 → 타 지자체와 공동대응 예정
- CCUS: 산자부·과기부와 재추진 협력

### AX·피지컬 AI
- 단순 데이터센터(AWS형) 유치 아닌 **AX 실증 연구센터 + 특화 SLM 개발** 지향
- 피지컬 AI 도입 시 숙련 노동자 데이터 생산 보상 문제 → 사회적 합의 필요
- **노정위원회 격상**: 노동특보 → 노정위원회 위원장, AX 시대 노사민정 거버넌스 강화

### 기타
- 생활임금 확대 검토, 배달노동자 쉼터 (편의점 협약 모델)
- 공업축제 전면 재검토: 처용문화제 등 전통문화 기반 전환 → 10월 행사 법적 리스크 검토
- 농수산물도매시장 이전 원점 재검토: 공론화위원회 구성 지시
- 울산항 동북아 에너지 물류 허브화: 러시아·일본·중국 대상 에너지 저장·수출 거점
- 산자부 지역특화산업: 울산만의 **상위 개념 융합 생태계 비전** 단일 패키지 기획 지시

## 교통국 (2026-06-17)

### 시내버스 정상화 (최우선 과제)
- 요금 인상(1,250원→1,500원) 후 이용객 감소, 보조금 오히려 160억 원 증가
- **버스기사 미적립 퇴직금 720억 원**, 통상임금 소송 예상비용 600~800억 원
- 3단계 해결: 단기(한정면허 복구·추경) → 중기(BTR·지능형체계) → 장기(교통공사 설립·공영제)

### 트램 1호선 재검토
- 실시설계 8월 완료 예정, 총사업비 3,814억 원 (재정사업)
- B/C: 2023년 0.85, 인건비·자재비 상승으로 현재 더 악화 → **별도 재평가 강력 지시**
- 연간 운영적자 수백억~천억 원 우려 (광주 2호선 참조)
- 공업탑 로터리 교통혼잡, 문수로 가로수 3km 재배치, 상권 영향 종합 검토 지시

### 산업단지 교통
- 새벽 3~4시 출근, 주차 없어 차 안에서 취침 → 횡단보도 미설치로 사망사고 발생
- 단기: 사선주차 확보 + 횡단보도 설치 / 중기: 유휴지 주차장 + 산단 셔틀 / 장기: DRT 연계

### KTX 태화강역
- 승강장 길이 연장(210m→412m) 국비 요구, 울산시 부담 312억 원 요구 → 중고속 열차 추가 정차 전략 수정 논의

### 울산공항
- 고도제한 완화: 타 지자체(강서구 등)와 연대 추진
- 공항 기능 조정(폐쇄 가능성) 장기 교통망 차원 검토 열어둠

## AI수도추진본부 (2026-06-18)

### 현황 진단
- 38명 조직, 대부분 행정직 파견 → **AI 기술 전문성 전무**
- 울산 기업 DX 현황 통계 없음 → AX 논의 전제 조건 부재
- 네이버·LG AI연구원·NHN 등 국내 LLM 기업과 협업 없음

### 전략 방향
- 기초 AI(LLM) 독자 개발 포기 → **조선·제조 분야 SLLM(소형거대언어모델) 집중**
- UNIST-현대중공업-판교 AI기업 컨소시엄: 조선 특화 파운데이션 모델 개발 착수 (70억)
- AWS 데이터센터: 자체 AI 생태계 없으면 전력 소모만 → SLM 기업 유치와 연결 필수
- 자율주행 버스: 현재 Rule-based(라이다 의존) 초보 수준, 테슬라 엔드투엔드 대비 뒤처짐

### 당선인 지시
1. **명칭 변경**: AI수도추진본부 → '산업AX추진단' (실질 반영)
2. **산업 AX 마스터 로드맵** 수립: 7월 초 초안, 7월 말 확정·발표
3. 로드맵 핵심: 기업별 DX 현황 파악 → SLLM 중소기업 보급 → AI 인재 타지역 유치
4. 조직 연결자 역할: 판교·서울·광주·UNIST 출장 통한 네트워크 확보

## 기업투자국 (2026-06-18)

### 펀드·투자
- 기존 13개 펀드 2,500억 + 지역성장펀드 500억(중소벤처부 공모) = **총 3,000억 원** (8월 출범)
- 자펀드 형태로 관내 벤처·스타트업 투자

### 창업 생태계
- **울산 내 AC·VC 인프라 극히 부족** → 창업자가 투자 못 받아 울산 이탈
- 서울 창업허브(공덕·마루360) 연계: 약 13평 서울 거점 구축 (2억 원, 4대 창업도시 사업비 활용)
- 글로벌: 도쿄·싱가포르 스타트업 생태계 연결
- 울산 스타트업 허브(종하이노베이션센터): 이용률 저조, 활성화 필요

### 소상공인
- 경영안정자금 1,100억 원 (전년 +300억)
- 울산사랑상품권(울산페이): 3,800억 발행, 캐시백 13%로 한시 확대 (4~8월)
- 공공배달앱 울산페달: 시장점유율 2%, 민간 독과점 속 부진

### 지방외교 재편
- 우호도시 교류를 경제 실리 중심으로 전면 재설정
- 러시아: 울산항 에너지 허브 연계, 천연가스 도시 우호관계 선제 구축
- 중국: 항저우(LLM 기술) 전략적 기술 교류
- 일본: 울산 석유 비축→일본 수출 구조 검토

## 경제자유구역청 (2026-06-18)

### 용지 문제
- 초기 지정이 기존 산단 중심 → 신규 분양 용지 부족
- 1차 추가 지정(KTX 역세권) 완료 / 2차 추가 지정 산업부 컨설팅 중
- 하이테크밸리(67만㎡) + KTX 복합특화지구(150만㎡) 개발 중

### 트램 B/C 재평가
- 교통국의 KDI 평가 수용 금지 → 인수위 지원단이 별도 재평가 지시
- 2023년 B/C 0.85 → 현재 인건비·자재비 상승으로 더 악화

### 직·주·락 복합 거점
- KTX역~하이테크밸리~UNIST 연결 구간: 산업·주거·문화 통합 개발 목표
