# 울산신용보증재단 사옥매입 논란

## 요약

울산신용보증재단을 둘러싼 논란은 **285억 원 규모 사옥 매입**에서 비롯된다. 재단은 독립 사옥이 장기 기관 운영과 소상공인 종합지원 기능 확대를 위해 필요하다는 입장이나, 인수위는 매입비 외에 주차장·리모델링 비용이 추가되면 재정 부담이 커질 수 있다고 우려했다. 재단은 이미 사옥매입지 주차타워 건립 타당성조사 및 기본계획 용역 최종보고서를 경영공시에 게시한 상태로, 부대시설 관련 추가 검토가 진행 중임을 시사한다.

## 외형 성과 (2026-03 보도 기준)

| 항목 | 내용 |
|---|---|
| 설립 시점 | 2000년 7월 |
| 보증잔액 | 1조 원 달성 (설립 26년 만) |
| 2026년 보증공급 목표 | 6,820억 원 |
| 2026년 신규 보증 목표 | 2,530억 원 |

| 전국 지역신보 평균 순사고율 | 4.98% |

## 사옥 매입이 문제가 되는 이유

신용보증재단은 소상공인·소기업 신용보증을 위해 유동성과 기본재산을 관리하는 정책금융기관이다. 재원이 현금·유가증권에서 부동산으로 이동하면 회계상 자산은 유지되지만 **즉시 사용 가능한 유동성·운용수익·보증 확대 여력·위기 대응력**이 약화될 수 있다는 것이 문제 제기의 핵심이다("건물을 사지 말자"가 아니라 "보증기관답게 운영되고 있는지 점검하자"는 취지).

| 위험 | 내용 |
|---|---|
| 유동성 저하 | 현금·유가증권 → 부동산 전환으로 즉시 사용 가능 재원 감소 |
| 운용수익 감소 | 유가증권 운용수익·이자수익 감소 가능 |
| 추가비용 발생 | 매입비 외 리모델링·주차장·유지관리·이전비 |
| 보증여력 논란 | 소상공인 지원 재원이 기관 자산 취득에 묶였다는 비판 |

## 핵심 재무 지표 (검증 필요 항목)

신용보증재단 재무상태의 핵심 지표는 기본재산·보증잔액·운용배수·사고율·대위변제율이다. **운용배수**는 보증잔액 ÷ (기본재산+이월이익금)으로, 서울시 신용보증재단 출연 검토자료 기준 지역신용보증재단의 보증 총한도는 기본재산과 이월이익금 합계의 **15배 이내**로 제한된다.

검증이 필요한 질문:
1. 사옥 매입 후 기본재산은 얼마나 유지되는가
2. 보증잔액 1조 원 대비 운용배수는 얼마인가
3. 사옥 매입으로 현금성 자산·유가증권은 얼마나 감소했는가
4. 운용수익 감소분은 얼마인가
5. 대위변제 증가 시 즉시 대응 가능한 유동성은 충분한가
6. 사옥 매입 후에도 2026년 보증공급 목표(6,820억 원)를 안정적으로 달성할 수 있는가

### 유동성 평가 기준표 (일반론)

**실질 유동성 자산 ÷ 보증잔액**

| 비율 | 해석 |
|---|---|
| 10% 이상 | 매우 안정적 |
| 7~10% | 대체로 안정적 |
| 5~7% | 관리 필요 |
| 3~5% | 취약 신호 |
| 3% 미만 | 유동성 위험 높음 |

**대위변제금 ÷ 실질 유동성 자산**

| 비율 | 해석 |
|---|---|
| 20% 이하 | 안정적 |
| 20~30% | 관리 가능 |
| 30~50% | 경계 |
| 50~70% | 취약 |
| 70% 초과 | 유동성 스트레스 큼 |

이 두 기준표는 신용보증재단 일반론(원 소스에 구체 산식 출처 불명확)이며, 울산신보의 실제 사옥매입 후 비율이 어느 구간에 해당하는지는 이 소스에 확정 수치로 제시되어 있지 않다. ^[ambiguous]
