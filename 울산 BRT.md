**BRT**는 **Bus Rapid Transit**, 한국어로 **간선급행버스체계**입니다. 
버스를 지하철처럼 빠르고 정시성 있게 운행하도록 만든 고급 버스 시스템입니다.

## 핵심 개념

BRT는 단순한 버스노선이 아니라 다음 요소를 묶은 체계입니다.

| 요소    | 내용                       |
| ----- | ------------------------ |
| 전용차로  | 일반 차량과 분리된 버스 전용 주행공간    |
| 중앙정류장 | 도로 중앙에 정류장을 설치해 정시성 확보   |
| 우선신호  | 교차로에서 버스가 먼저 통과하도록 신호 제어 |
| 환승시설  | 버스·철도·자전거·보행과 쉽게 연결      |
| 고급 차량 | 대용량·저상·전기·수소버스 등 활용 가능   |
| 통합관제  | 배차·혼잡·운행시간 관리            |

국토교통부도 BRT를 전용주행로, 입체교차로, 정류소, 전용차량 등을 갖추어 급행으로 버스를 운행하는 교통체계로 설명합니다. ([molit.go.kr](https://www.molit.go.kr/mtc/USR/WPGE0201/m_36794/DTL.jsp?utm_source=chatgpt.com))

## 일반 버스와 차이

| 구분  | 일반 시내버스      | BRT        |
| --- | ------------ | ---------- |
| 주행  | 일반 차량과 섞여 운행 | 전용차로 중심    |
| 속도  | 교통체증 영향 큼    | 상대적으로 빠름   |
| 정시성 | 낮음           | 높음         |
| 정류장 | 도로변 정류장      | 중앙정류장·환승거점 |
| 신호  | 일반 신호 따름     | 버스 우선신호 가능 |
| 기능  | 노선별 이동수단     | 도시 간선교통축   |

중요한 점은 **중앙버스전용차로 = BRT**가 아니라는 것입니다. 
중앙버스전용차로는 BRT의 일부일 뿐이고, 
BRT는 전용차로·우선신호·환승센터·운행관리까지 묶어야 합니다. 서울시 설명도 중앙버스전용차로에 환승시설, 교차로 우선처리, 전용차량, 첨단 운영·관제시스템 등이 결합된 것이 BRT라고 설명합니다. ([mediahub.seoul.go.kr](https://mediahub.seoul.go.kr/archives/2003163?utm_source=chatgpt.com))

## 울산에 적용하면

울산에서는 BRT가 의미 있는 이유가 있습니다. 울산은 도시철도가 없었고, 시내버스 의존도가 높은데도 버스 신뢰도가 낮다는 문제가 있습니다. 울산시는 이런 열악한 대중교통 환경을 극복하기 위해 트램 건설을 추진하고 있습니다. ([ulsan.go.kr](https://ulsan.go.kr/u/traffic/contents.ulsan?mId=001006004002000000&utm_source=chatgpt.com))

그런데 트램은 건설비와 시간이 많이 듭니다. 그래서 BRT는 다음과 같은 역할을 할 수 있습니다.

|역할|설명|
|---|---|
|트램 전 단계|트램이 들어오기 전 간선 교통축 보강|
|트램 보완|트램 노선이 닿지 않는 구간을 빠른 버스로 연결|
|저비용 간선|철도보다 싸고 빠르게 구축 가능|
|수소버스 활용|울산 수소산업과 연계 가능|
|DRT 연결|외곽 DRT를 BRT 정류장에 붙이는 구조 가능|

## 울산형 구상

울산에서 BRT를 한다면 단순히 “버스 빠르게”가 아니라 다음 조합이 좋습니다.

```text
외곽·저수요 지역: DRT
→ 생활권 거점: 환승센터
→ 도심 간선축: BRT
→ 장래 핵심축: 트램
```

즉, **DRT는 모세혈관**, **BRT는 간선도로**, **트램은 도시철도 축**으로 나누는 방식입니다.

## 정책 판단

울산에서 BRT를 검토할 때 핵심 쟁점은 세 가지입니다.

|쟁점|질문|
|---|---|
|도로공간 재배분|일반차로를 줄이고 버스전용차로를 만들 수 있는가|
|수요 확보|버스 이용자가 실제로 늘어날 만큼 빠르고 편리한가|
|트램과 관계|트램과 중복되는가, 아니면 보완하는가|

결론적으로, **BRT는 울산의 버스체계를 단순 노선망에서 “간선 대중교통망”으로 바꾸는 수단**입니다. 울산에서는 트램만으로 모든 생활권을 커버하기 어렵기 때문에, **트램 + BRT + DRT**를 함께 설계하는 것이 가장 현실적인 방향입니다.
